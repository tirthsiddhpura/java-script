JavaScript Variable – Examples / Exercise

Example 1:

console.log(nm); let nm = "name";

Answer or Error: ❌ ReferenceError

Why:
let variable hoisted hota hai lekin use karne se pehle access nahi hota (Temporal Dead Zone). Isliye error aata hai.

Example 2:

console.log(b); var b = "username";

Answer or Error: ✅ Output = undefined

Why:
var hoisted hota hai. Declaration upar chali jaati hai, lekin value baad me assign hoti hai.

Example 3:

var x = 1; { var x = 2; } console.log(x);

Answer or Error: ✅ Output = 2

Why:
var block scoped nahi hota. Dono jagah same variable hota hai, isliye value overwrite ho jaati hai.

Example 4:

let a = 10; { let a = 20; console.log("Inside:", a); } console.log("Outside:", a);

Answer or Error: ✅ Output

Inside: 20 Outside: 10

Why:
let block scoped hota hai. Andar aur bahar alag-alag variables bante hain.

Example 5:

const person = { name: "demo" }; person.name = "user"; // ✅ person = {}; // ❌

Answer or Error:

  • person.name = "user" → ✅ No Error
  • person = {} → ❌ TypeError

Why:
const me object ka reference change nahi hota, lekin uski properties change ho sakti hain.
